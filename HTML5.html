META 요소


문자 세트 정의 : <meta charset="UTF-8">

웹페이지에 대한 설명 : <meta name="description" content=" ">

검색엔진에 대한 키워드 : <meta name="keyword" content=" ">

페이지 작성자 정보 : <meta name="author" content=" ">

뷰포트 설정 : <meta name="viewport" content="width=device-width, initial-scale=1.0, user-scalable=no">

LINK 요소


외부 스타일 시트에 연결하는 데 사용 : <link rel="stylesheet" type="" href="">

미리 파일을 로드하는 경우 : <link rel="preload" as="image" href="">

autoplay : 자동 재생
controls : 컨트롤 막대 표시
loop : 반복 재생
muted : 음소거
preload : 재생 버튼을 누르기 전 파일을 미리 다운로드
•none : 재생 버튼을 눌러야만 다운로드 하기 시작
•metadata : 파일 전체를 다운로드 하지 않고 메타 정보만 다운로드
•auto(기본값) : 즉시 이용할수 있도록 문서를 로드할 때마다 파일도 모두 다운로드
