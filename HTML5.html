폼을 사용하면 사용자가 데이터를 입력하여 송신하는 시스템을 마련할 수 있음
form 요소는 해당 범위가 입력폼이라는 것을 나타냄
(폼에 입력된 데이터를 실제로 보낼 때는 송신된 데이터를 처리하기 위한 프로그램(ex PHP)이 필요함)

<form action="데이터를 보낼 곳 URL"> ~ </form>

•동작 속성 : action - 폼에 입력한 데이터를 처리하는 프로그램의 URL을 지정
•대상 속성 : target
•메소드 속성 : method - 데이터의 송신 형식
  (get:기본값, URL 다음에 ?와 폼에이터 추가하여 송신 / post : 폼데이터만 본문으로 송신)
•name 속성 : 서버에 전송되는 field name (cf fomr 자체의 이름은 id)

input 요소의 type 속성을 지정하여 입력폼의 부품을 만듦

<input type=" "> ~ </input>
