태그와 속성

    <열기태그 속성="속성: 값;"> 내용 </닫기태그>

    속성의 값은 ;(세미콜론)으로 단위 구분

    속성 안에(따옴표 안에) 따옴표를 쓰고 싶으면 : " '내용' " 또는 ' "내용"'

    1. <a href="">

    2. <img src=""> <img width=""> <img height=""> <img alt="">:대체 텍스트 지정

    3. <html lang="">

    4. <p title>:단락이름 지정(툴팁박스)

    5. style=""

    문서 유형 선언 : HTML5

  • <!doctype html>

  • 문법 검사 : http://validator.w3.org

      META 요소

    1. 문자 세트 정의 : <meta charset="UTF-8">

    2. 웹페이지에 대한 설명 : <meta name="description" content=" ">

    3. 검색엔진에 대한 키워드 : <meta name="keyword" content=" ">

    4. 페이지 작성자 정보 : <meta name="author" content=" ">

    5. 뷰포트 설정 : <meta name="viewport" content="width=device-width, initial-scale=1.0, user-scalable=no">

      LINK 요소

    1. 외부 스타일 시트에 연결하는 데 사용 : <link rel="stylesheet" type="" href="">

    2. 미리 파일을 로드하는 경우 : <link rel="preload" as="image" href="">

      서식 요소 : 특수한 유형의 텍스트를 표시

    1. <hr> : 수평선(구분선)

    2. <b> 굵은 텍스트 vs <strong> 중요한 텍스트

    3. <i> 이탤릭체 vs <em> 강조된 텍스트

    4. <mark> 표시된 텍스트

    5. <small> 작은 텍스트

    6. <del> 삭제된 텍스트 <ins> 삽입된 텍스트

    7. <sub> 아래 첨자 <sup> 위 첨자

      목록 요소

    1. <ul> : unorderd list - <li>
    2. <ol> : orderd list - <li>
    3. <dl> : 설명목록을 정의 - <dt> <dd>

      그룹핑 요소

    1. <div> : block 레벨
    2. <span> : inline 레벨

    <pre> : 사전 형식화 된 텍스트를 정의
