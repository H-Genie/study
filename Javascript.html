<프로그래밍 언어 ≒ 인간의 행동>

1. 정보를 기억 (Memory)
- 인간 : 주변에 있는 사람을 기억
- 언어 : 변수를 선언

2. 조건을 판단 (Condition)
- 인간 : 이 사람이 엄마이고, 이 사람이 아빠인지 판단
- 언어 : 조건문 (조건에 맞는 명령을 실행)

3. 정보를 계산 (Operator)
- 인간 : 백원짜리와 만원짜리가 같은지 계산
- 언어 : 산술을 연산

4. 행동 (Act)
- 인간 : 아빠와 돈을 바꿈
- 언어 : 명령문 ⇒ 블럭으로 모이게 되면 함수

※ 가족은 Object : "아빠는 누구고, 엄마는 누구고~"가 모여서 데이터가 됨
